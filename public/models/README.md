# Face Recognition Models

Please download the following models from the face-api.js repository and place them in this folder:

Download from: https://github.com/justadudewhohacks/face-api.js/tree/master/weights

Required models:
1. tiny_face_detector_model-weights_manifest.json
2. tiny_face_detector_model-shard1
3. face_landmark_68_model-weights_manifest.json
4. face_landmark_68_model-shard1
5. face_recognition_model-weights_manifest.json
6. face_recognition_model-shard1
7. face_recognition_model-shard2

Or run this command to download them automatically:
```bash
npx degit justadudewhohacks/face-api.js/weights public/models
```
